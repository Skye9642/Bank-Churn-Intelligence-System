# 🏦 AI-Powered Customer Churn Intelligence System

## 📌 Project Overview
This project predicts customer churn in European retail banking using machine learning and explainable AI techniques. The system assigns churn probability scores and provides risk segmentation for proactive retention strategies.

---

## 🚀 Features
- Predictive churn risk scoring (0–100%)
- Random Forest classification model
- SHAP-based explainable AI analysis
- Scenario-based risk simulation
- Business retention recommendations
- Interactive Streamlit dashboard

---

## 🛠 Technologies Used
- Python
- Scikit-learn
- SHAP
- Pandas
- NumPy
- Streamlit
- Matplotlib

---

## ▶️ How to Run Locally

1. Install required libraries:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

---

## 📊 Business Impact
This system enables proactive churn management, targeted retention campaigns, and improved customer lifetime value using predictive and prescriptive analytics.
